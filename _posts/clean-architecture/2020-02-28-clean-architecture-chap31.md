# 31. 웹은 세부사항이다

## 끝없이 반복하는 추

연산 능력을 중앙에 집중하는 방식과 분산하는 방식 사이에서 끝없이 움직인다.

애플리케이션을 보호하기 위해 업무 규칙을 분리해야 함

## 요약

GUI는 세부사항. 웹은 GUI. 따라서 웹은 세부사항
